# **App Name**: Triumph Over Vice

## Core Features:

- Category Selection: Allow users to select their addiction category: Porn, Alcohol, or Smoking.
- Smooth Onboarding: Design a flowing signup process, enabling seamless user registration.
- Daily Progress Logging: Prompt users daily to log their progress in overcoming their addiction.
- Stats Keeping: Calculate and visualize stats about user progress, tracked daily.
- Goal setting: User must have the ability to set and revise a long term goal

## Style Guidelines:

- Primary color: Calm blue (#77B5FE), offering a sense of serenity and hope. 
- Background color: Light blue (#D8E6F7), a very low saturation shade of the primary color that gives the UI a clean, soothing background.
- Accent color: Gentle purple (#9D7DFE), for actionable items, providing contrast while maintaining the app's peaceful theme.
- Font: 'PT Sans', a clear and accessible sans-serif, suitable for all text and reflective of the app's support.
- Use minimalistic icons that are consistent with Apple iOS styling guidelines.
- Follow Apple iOS UI standards for tab bars, navigation, and modal presentations to ensure a familiar user experience.
- Incorporate subtle, encouraging animations for completing daily logs and achieving milestones.
